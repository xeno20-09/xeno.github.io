E-congé

Application Web

Projet : e-Congé

  • Catégorie : Application web
  • Client : Cour Suprême
  • Technologies utilisées : Laravel 10 · MySQL · Laravel DOMPDF · Laravel/UI · Maatwebsite Excel · Spatie Laravel Permission
  • Accès : Application interne sécurisée (confidentielle, non publiée en ligne)

Application de gestion automatisée des congés

e-Congé est une application développée pour la Cour Suprême afin de simplifier la gestion interne des congés du personnel. Elle centralise tout le processus au niveau du service RH, qui saisit les informations du personnel, le type de congé choisi et vérifie automatiquement le solde congé disponible.

Si toutes les conditions sont respectées, la demande est validée et une lettre officielle est générée puis envoyée au personnel par e-mail professionnel. Le personnel peut ensuite consulter son espace pour visualiser ses informations : solde de congés, historique et statut de ses demandes.

Fonctionnalités principales

  • ✅ Interface RH intuitive avec formulaire de saisie rapide
  • ✅ Vérification automatique du solde de congé avant validation
  • ✅ Génération et envoi automatique de la lettre de congé par e-mail
  • ✅ Espace personnel pour le suivi des congés et du solde restant
