eSchool

Application Web

Projet : eSchool (cas d'ecole Primaire)

  • Catégorie : Application web
  • Client : Projet personnel
  • Technologies : Symfony · MySQL · Bootstrap · Twig
  • Type de projet : Projet personnel

Application de gestion scolaire complète

eSchool (cas d'ecole Primaire) est une solution web permettant de gérer efficacement les établissements scolaires : élèves, enseignants, cours et notes. Elle offre une interface simple et intuitive pour centraliser toutes les informations académiques et administratives.

Fonctionnalités principales

  • ✅ Gestion des élèves, enseignants , cours et classes
  • ✅ Enregistrement et consultation des notes
  • ✅ Tableau de bord pour administrateurs et enseignants
  • ✅ Génération de relevés de notes
  • ✅ Gestion des scolarités
