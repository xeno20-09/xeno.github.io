Projet : QuizSport

  • Catégorie : Application web
  • Client : Projet personnel
  • Technologies : Laravel · MySQL · Bootstrap · OpenTriviaDB
  • Type de projet : Projet personnel

Application de quiz sportif interactif

QuizSport est une application web permettant aux utilisateurs de tester leurs connaissances sportives via des quiz interactifs. Chaque utilisateur peut participer à différents quiz, voir ses scores et suivre son évolution au fil du temps.

Les questions sont rédigées en anglais

Fonctionnalités principales

  • ✅ Création et gestion de quiz dynamiques
  • ✅ Interface ludique et responsive pour les joueurs
  • ✅ Suivi des scores et statistiques par le joueur
